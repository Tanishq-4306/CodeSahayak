## 🔧 TROUBLESHOOTING

### Backend Not Running?
```bash
# Check status:
Open: test-complete-system.html
Look at: Backend Status

# If not running:
cd backend
node server.js
```

### Frontend Not Loading?
```bash
# Use frontend server:
START_FRONTEND.bat

# Then open:
http://localhost:8080/main.html
```

### Test Dashboard Not Opening?
```bash
# Open manually in browser:
Right-click: test-complete-system.html
Select: Open with > Browser
```

### Database Errors?
```bash
# Restart backend:
Stop: Ctrl+C in backend terminal
Start: node backend/server.js
```
