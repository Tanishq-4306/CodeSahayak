## 🔧 Customization

### Change Colors
Edit `css/premium-enhancements.css`:
```css
:root {
    --primary-blue: #YOUR_COLOR;
    --primary-teal: #YOUR_COLOR;
    --accent-orange: #YOUR_COLOR;
}
```

### Adjust Animations
Edit `js/premium-animations.js`:
```javascript
const duration = 2000; // Change animation speed
const threshold = 0.1; // Change trigger point
```
